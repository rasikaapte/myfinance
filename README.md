# MyFinance

A modern React web application to help you track and manage your expenses.

## Features

- **Add Expenses**: Record expenses with amount, category, description, and date
- **View Expenses**: See your transactions grouped by date
- **Expense Summaries**: Get insights by time period (this month, last month, this year, all time)
- **Categories**: 8 colorful categories with icons (Food, Transportation, Shopping, etc.)
- **Delete Expenses**: Remove incorrect entries with a click
- **Persistent Storage**: Data stored in browser localStorage
- **Beautiful UI**: Modern, responsive design with smooth animations

## Getting Started

### Prerequisites

- Node.js 18 or higher
- npm or yarn

### Installation

1. Clone the repository:
```bash
git clone https://github.com/rasikaapte/myfinance.git
cd myfinance
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm run dev
```

4. Open http://localhost:5173 in your browser

### Build for Production

```bash
npm run build
npm run preview
```

## Tech Stack

- **React 18** - UI library
- **Vite** - Build tool and dev server
- **localStorage** - Data persistence
